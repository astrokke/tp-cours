Créer une API qui permet de gérer une liste de tâches. Les utilisateurs peuvent ajouter, mettre à jour, récupérer et supprimer des tâches à l'aide des méthodes HTTP GET, POST, PUT et DELETE.

On utilisera le no sql

Une tâche est composé de :
titre string required,
description string,
date_debut : date,
date_fin : date,
done : boolean
